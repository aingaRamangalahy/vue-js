# JavaScript & TypeScript Through Vue.js Core

This site hosts a **numbered lesson series** on JavaScript and TypeScript, grounded in real examples from the Vue.js ecosystem.

::: tip Where to start
Open the **[course roadmap](/javascript-course/)** for the full learning path and links to every lesson.
:::

## Local development

```bash
npm install
npm run dev
```

Build static output to `docs/.vitepress/dist`:

```bash
npm run build
```
